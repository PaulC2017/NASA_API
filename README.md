### Getting Started

#### Prerequisites

- [Node.js](https://nodejs.org/en/download/)

#### Installation

```bash
git clone https://github.com/PaulC2017/NASA_API.git
cd NASA_API
npm install
```

#### Usage

```bash
npm start
```

### Architecture

- Client-side JavaScript, served by GitHub pages.
